# Bill Tracker

## Software Engineering

### Description

This project is a web application that does stuff!

### Prerequisites

Verify Node installation
```bash
node -v
npm -v
```

[Install Node](https://nodejs.org/en/download/package-manager)

Install yarn
```bash
npm install --global yarn
```

### Running the API

Install yarn
```bash
cd server
./run.sh

```

### Running the frontend

```bash
cd ../frontend
yarn
yarn dev
```
